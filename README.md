# gimme-a-spam
 Node.js로 작성한 [스티비 스팸월드컵](http://recruit.stibee.com/) 예선문제 답안


## Requirements
Node.js >= 4 혹은 iojs

## Getting Started
```bash
$ git clone https://github.com/mooyoul/gimme-a-spam
$ cd gimme-a-spam
$ npm install
$ env MANDRILL_API_KEY=YOUR_API_KEY node .
$ # or
$ # env MANDRILL_API_KEY=YOUR_API_KEY node index.js
```
